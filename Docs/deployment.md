Ho eseguito il comando git clone <URL del repository> per clonare il repository in locale.
Ho creato un nuovo branch con il comando git checkout -b feature-martinatuveri.
Ho utilizzato il comando mkdir project-cms per creare una nuova directory chiamata "project-cms" all'interno del repository clonato.
Mi sono spostata nella nuova directory eseguendo il comando cd project-cms.
Ho creato un nuovo file chiamato docker-compose.yml per definire i servizi e le dipendenze del mio ambiente containerizzato. 
Ho configurato i servizi Docker nel file docker-compose.yml includendo le immagini di WordPress e MySQL, impostando le variabili d'ambiente e le dipendenze necessarie.
Ho avviato il container con il comando docker compose up -d
